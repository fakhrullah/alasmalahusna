Di bina oleh Fakhrullah Padzil di bawah lesen ... .
tolong saya, saya nak lepaskan bawah Creative Commons Zero (CC0) tapi web ni guna Reveal.js(MIT) dan guna gulp (MIT).

Saya juga mengajak anda untuk membantu saya membetulkan dan menyediakan slaid ini dengan fork saya di Github.
